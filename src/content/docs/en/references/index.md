## Articles

- VRChat Official Documentation: [VRChat](https://docs.vrchat.com/)

Found the official docs later. It's all in English; I mostly read through it using machine translation.

- Blender Official Documentation: [Blender 2.93 Reference Manual](https://docs.blender.org/manual/en/2.93/)

As we all know, official docs are always impossible to understand.

- [Unity User Manual (2019.4 LTS)](https://docs.unity3d.com/2019.4/Documentation/Manual/index.html)

- [VRChat Model Creation and Upload Overview (201912) - Xingkong Yue Ling -博客园](https://www.cnblogs.com/raitorei/p/12015876.html)

The first article I read. It was the top result when Googling `vrchat upload mmd model`. Very comprehensive, but the tutorial is a bit dated.

- [Unity Animator State Machine - 1024Sou](https://www.1024sou.com/article/17901.html)

Covers various Unity animation features ~~(though I didn't understand the last part)~~.

- [The Role of Write Defaults - CSDN Blog](https://blog.csdn.net/yjy99yjy999/article/details/82730948)

Reference used for the `VRLabs Marker` section.

- [Blender - 2.8x - Screencast Keys (v3.0) - Plugin Introduction and Installation](https://blog.csdn.net/linjf520/article/details/108108603)

Plugin reference for displaying keypresses in the Blender head-swapping video.

- [【Notes】Dynamic Bone (& Jiggle) Research Notes](https://forum.gamer.com.tw/Co.php?bsn=60602&sn=5966)

~~I didn't understand it, but I was deeply impressed.~~ Three articles on adding Dynamic Bone collisions.

- [Detailed Explanation of Unity Animation System 3: How to Play and Switch Animations? - Zhihu](https://zhuanlan.zhihu.com/p/144912096)

Finally understood the parts I didn't get before!

- [Unity Cloth Simulation Tutorial](https://www.cg.com.tw/UnityCloth/)

- [Introduction to Simple and Common Unity Particle System Functions - Tencent Games Institute](https://gameinstitute.qq.com/community/detail/126886)

- [Installing a Specific Version of Unity3D on Linux - CSDN Blog](https://blog.csdn.net/OOFFrankDura/article/details/123487279)

- [How to Delete and Merge Overlapping/Adjacent Points in Blender 2.92 - Baidu Experience](https://jingyan.baidu.com/article/36d6ed1f59c1e55ace488348.html)
