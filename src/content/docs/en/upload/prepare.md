> The road from VRChat beginner to Unity master begins here! XD

## 1. Requirements

### 1. Have a VRChat account registered on the official website

A Steam-only login will not work. If your name has extra characters after it, that means you are using Steam login. Please register an account on the official website and link it.

### 2. Reach the blue account rank, New User

Online time, often visiting crowded worlds, the number of friends you have, and spending time with friends can all help you rank up faster.

## 2. Download and Install Unity

### 1. Open the Unity download page

Unity download page (CN): [All Unity versions - Unity China](https://unity.cn/releases)

> If possible, I recommend downloading the international version, but the China version can also be used.

### 2. Download and install Unity Hub

You need to register and log in before downloading Unity.

### 3. Download Unity 2019.4.31

Downloading through Unity Hub makes it easier to install the Chinese language pack. If you forget to select it, that is fine; a later article includes a method for that.

The Unity version currently supported by the VRChat SDK is 2019.4.31f1. You can check it here: [Currently Supported Unity Version](https://docs.vrchat.com/docs/current-unity-version)

## 3. Download the VRChat SDK

### 1. Open the VRChat SDK download page

VRChat download page: [Download - VRChat](https://vrchat.com/home/download)

### 2. Download SDK 3.0

Click “Download SDK3 - Avatars” to start the download.  
Note: some older avatars use SDK2, so you may need to download SDK2 from “Other”. However, I recommend upgrading to SDK3 before uploading. A later article introduces this.

## 4. Download Required Plugins

### 1. Dynamic Bone plugin

Download: [Dynamic Bone 1.2.0](https://raw.githubusercontent.com/yexca-VRChat/vrchat-docs-source/main/file/Dynamic%20Bone%201.2.0.zip)

### 2. Shader plugin

Download: [UTS2](https://raw.githubusercontent.com/yexca-VRChat/vrchat-docs-source/main/file/UTS2.zip)
