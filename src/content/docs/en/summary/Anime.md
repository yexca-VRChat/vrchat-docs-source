First, let's talk about making animations. Usually, it's just about toggling an object on or off.

## Toggles

An object toggle means recording an animation to enable or disable an item.
You can parent these objects to body parts (bone parenting), such as your hands.

Particle effects can also be triggered by recording a toggle.

## Animations

General animations are usually MMD dance moves or poses. I recommend using pre-made ones, as making them yourself is quite difficult.

### Ready-made

For MMD dances, you can typically use tools to convert them into files that Unity can recognize. For poses, there are usually finished assets available for purchase.

### DIY

Making these directly in Unity is impractical (you can find properties under the Animation window -> Add Property -> Animator). It's better to create skeletal animations in other software and then import them into Unity.

## Others

This includes things like moving objects, facial expressions, or changing shaders to swap hair colors, etc.
