完成品モデルっていうのは、他の人がほとんど全部作ってくれてるモデルのこと。自分はアップロードするだけでOK。
大体は [Booth](https://booth.pm/ja) で買えるよ。

## 一、Unity の言語設定（簡体字中国語化）の準備

### 1）Unity Hub から簡体字中国語コンポーネントをインストールする

Unity Hub を開いてログインし、左側の `インストール` をクリック。`2019.4.31f1c1` (c1 は中国版を指す) を見つけて、右側の歯車アイコンをクリック。`モジュールを加える` を選択して、`日本語`（または元のテキスト通りなら `简体中文`）を見つけて選択し、インストールする。

### 2）「モジュールを加える」がない、または Unity Hub からインストールしていない場合

注：この方法は試したことがないから、使える保証はないよ。

#### <1> zh-cn.po ファイルをダウンロードする

ダウンロードリンク：[zh-cn.po](https://raw.githubusercontent.com/yexca-VRChat/vrchat-docs-source/main/file/zh-cn.zip)

#### <2> ファイルを Unity のインストールディレクトリに置く

```markdown
C:\Program Files\Unity\2019.4.31f1c1\Editor\Data\Localization
```

もしこのディレクトリがなかったら、自分で作ってみて。

## 二、Unity プロジェクトの新規作成と言語設定

### 1）個人ライセンスを追加する

左上のアイコンから「ライセンスを管理」をクリック。「加える - 無料の個人用ライセンスを取得 - 同意して個人用ライセンスを取得」を選択すればOK。

### 2）Unity プロジェクトを新規作成する

「新しいプロジェクト」をクリックして「3D」を選択。プロジェクトに名前をつけて、保存場所を選んだら「プロジェクトを作成」をクリック。しばらく待つよ。

### 3）言語設定

Unity を深く学びたいなら、英語の原版のまま使うのがおすすめ。

「Editor - Preferences - Languages」の下で「简体中文」（または日本語）を選択する。

## 三、ファイルのインポート (unitypackage)

ファイルのインポートは、ダブルクリックするか、直接 Unity にドラッグ＆ドロップすればできるよ。

### インポートが必要なファイル

1）Dynamic Bone プラグイン (Dynamic Bone 1.2.0.unitypackage)

2）シェーダープラグイン (UTS2.unitypackage)

3）VRCSDK (VRCSDK3-AVATAR-2022.02.16.19.13_Public.unitypackage ※新しい SDK が出てるから、名前はこれじゃないかも)

4）モデルファイル

## 四、モデルのアップロード

### 1）モデルを配置する

下の `Assets` の中から prefab ファイルかシーンファイルを見つけるよ。

#### <1> prefab ファイルの場合

左側のヒエラルキー（シーン）にドラッグ＆ドロップする。

#### <2> シーンファイルの場合

ダブルクリックして開く。

### 2）VRC コントロールパネルを開く

上の `VRChat SDK - Show Control Panel` をクリック。VRChat アカウントでログインしたら、`Builder` をクリックする。

#### <1> 下の “Build & Publish for Windows” がクリックできる場合

そのままクリックして、しばらく待てばOK。

#### <2> クリックできない場合

上に六角形の赤いビックリマークが出ているはず。それぞれの後ろに `Auto Fix` があるから、全部クリックしてから `Build & Publish for Windows` を押してみて。
もし全部押してもアップロードできない場合は…ごめん、まだ遭遇したことがないな。

#### <3> モデル情報が出ず、"A VRCAvatarDescriptor is required to build an Avatar" と表示される場合

この場合は、アップロードしようとしているモデルが SDK2 のモデルってこと。新しくプロジェクトを作って SDK2 をインポートしてアップロードするか、SDK3 モデルにアップグレードしてね（後の記事で書く予定。いつか SDK2 がサポートされなくなるかもしれないから、アップグレードがおすすめ）。

### 3）モデル情報を入力する

作成に成功したら、`VRChat アップロードパネル` は閉じても大丈夫。

Avatar Name はモデルの名前。

Description はモデルの説明。

次の警告は無視してOK。

Sharing はアップロードするモデルを公開するかどうか。`Private` は非公開、`Publish`（Public）は公開（他の人にコピーを許可するかどうか。普通は Private を選ぶよ）。

一番下のチェックボックスにチェックを入れたら、`Upload` をクリックしてアップロード完了！

### 4）モデルのサムネイル（表紙）を作る

上の手順だけでアップロードできるけど、右側の画像が気に入らないこともあるよね。

#### <1> オブジェクトを作成する

シーンに戻って、左側の `VRCCam` の上で右クリックして `3D オブジェクト - Quad` を選択。

右側の `Transform` で、`位置 (Position)` の z 軸を `2.6` に、`拡大縮小 (Scale)` の x 軸を `4`、y 軸を `3` に設定する。

#### <2> 画像を追加する

アップロードしたい画像を見つけて Unity にドラッグ＆ドロップ。次にマテリアルを作成して（下の `Assets` で右クリック `作成 - マテリアル`）、その `マテリアル` をクリック。右側の `Shader` を選択（例：`VRChat-Mobile-Toon Lit`）。それから画像を右側のマテリアルのスロットにドラッグして、最後にそのマテリアルをシーン上のさっき作った Quad にドラッグ＆ドロップする。

#### <3> モデルをアップロードする

このままだとモデルが画像を遮ってしまうから、左側でモデルを選択して、右側のモデル名の横（左側）にあるチェックボックスを外してモデルを非表示にする。

それから `Game` 画面に戻ると、右側にさっき作った画像が見えるはず。

この状態でもう一度 `Upload` をクリックすればOK。

## 五、エラーが発生してアップロードに失敗したら

> 再起動で 90% の問題は解決するし、再インストールで 99% 解決する。買い替えれば 100% 解決だね。

もう一度アップロードしてみる、Unity を開き直す、新しいプロジェクトを作ってアップロードし直す、パソコンを再起動する、ネットワーク環境を確認する。

もしブループリントの問題（エラーメッセージに `BluePrint` がある場合）は、左側でキャラクターを選択して、右側の一番下までスクロールして `Detach` をクリックしてから、もう一度アップロードしてみて。
