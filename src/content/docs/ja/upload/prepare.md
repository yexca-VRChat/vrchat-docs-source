> VRChat 入門から Unity マスターへの道が始まります！XD

## 1. 前提条件

### 1. VRChat 公式サイトで登録したアカウントを持っている

Steam ログインだけでは使えません。名前の後ろに別の文字が付いている場合は Steam ログインです。公式サイトでアカウントを登録し、連携してください。

### 2. アカウントランクが青色 (New User) になっている

オンライン時間、人の多い場所によく行くこと、フレンド数、フレンドと一緒にいることなどでランクアップが早くなります。

## 2. Unity のダウンロードとインストール

### 1. Unity 公式のダウンロードページを開く

公式ダウンロードページ (CN): [Unity すべてのバージョン - Unity China](https://unity.cn/releases)

> 可能なら国際版をおすすめしますが、中国版でも使えないわけではありません。

### 2. Unity Hub をダウンロードしてインストールする

Unity をダウンロードするには、先に登録してログインする必要があります。

### 3. Unity 2019.4.31 をダウンロードする

Unity Hub からダウンロードすると、中国語化を選びやすくなります。選び忘れても大丈夫です。後の記事で方法を紹介します。

現在 VRChat SDK が対応している Unity バージョンは 2019.4.31f1 です。こちらで確認できます: [Currently Supported Unity Version](https://docs.vrchat.com/docs/current-unity-version)

## 3. VRChat SDK のダウンロード

### 1. VRChat SDK のダウンロードページを開く

VRChat ダウンロードページ: [Download - VRChat](https://vrchat.com/home/download)

### 2. SDK 3.0 をダウンロードする

“Download SDK3 - Avatars” をクリックするとダウンロードが始まります。  
注意: 一部の古いアバターは SDK2 用なので、“Other” から SDK2 をダウンロードする必要があるかもしれません。ただし、SDK3 にアップグレードしてからアップロードすることをおすすめします。後の記事で紹介します。

## 4. 必要なプラグインのダウンロード

### 1. Dynamic Bone プラグイン

ダウンロード: [Dynamic Bone 1.2.0](https://raw.githubusercontent.com/yexca-VRChat/vrchat-docs-source/main/file/Dynamic%20Bone%201.2.0.zip)

### 2. シェーダープラグイン

ダウンロード: [UTS2](https://raw.githubusercontent.com/yexca-VRChat/vrchat-docs-source/main/file/UTS2.zip)

### 3. その他

その他のプラグインは自分で探してみてください。例: [VRC 工具箱](https://www.bilibili.com/video/BV13q4y1f7ZJ)
