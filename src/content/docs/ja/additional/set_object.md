> ワールドにオブジェクトを置く方法だよ。

## 一、プレハブを使う方法

### 1）ダウンロードして Unity にインポートする

ここからダウンロードしてね：[安定剛体](https://raw.githubusercontent.com/yexca-VRChat/vrchat-docs-source/main/file/稳定刚体.zip)

ダウンロードした後に解凍して、フォルダごと Unity にドラッグ＆ドロップしよう。

### 2）セットアップ

prefab（プレハブ）ファイルを見つけて、アバター（モデル）の直下にドラッグ＆ドロップする。その後、右クリックして「Unpack Prefab（プレハブを完全に展開）」を選択してね。

その中にある「Anchor」を1つ上の階層（モデルの直下）に移動させる。名前は自由に変えてOK（例：「ベナレス」）。

次に「worldSpace」->「Container」->「Cube」の順に展開していき、「Cube」についている「Mesh Renderer」コンポーネントを削除（Remove）する。そのあと、ワールドに置きたいアバター（モデル）を「Cube」の配下に移動させよう。

「worldSpace」の名前も自由に変更して大丈夫だよ（例：「ベナレス\_Model」）。

### 3）アニメーションの録画（レコーディング）

「worldSpace」（この例では「ベナレス\_Model」）をアクティブ（ON）にするアニメーションを1つ作成するだけでOK。

## 二、プレハブを使わない方法

### 1）空のオブジェクトを作成する

まず、モデルの位置（Position）を「0, 0, 0」にする。そのあと、モデルの配下に空のゲームオブジェクト（右クリック -> 「Create Empty」）を作成してね。

さらに、作成した空のオブジェクトの配下に、5つの空のオブジェクトを作成する（1つ作ってから「Ctrl + D」で複製すると楽だよ）。

これら5つの空のオブジェクトの名前を、それぞれ「世界位置源」「更新位置源」「世界位置」「更新位置」「オブジェクト」に変更する。そして「更新位置」を「世界位置」の配下に入れる。

その後、モデル（置きたいアイテム）を「オブジェクト」の配下に移動させる。モデル自体は自由に動かして位置調整してもいいけど、「オブジェクト」自体は動かしちゃダメ。位置（Position）は「0, 0, 0」のままにしておいてね。

### 2）コンストレイント（Constraint）の追加

#### <1> 「世界位置」のコンストレイント

「世界位置」を選択して、インスペクター（右側）で「Position Constraint」と「Rotation Constraint」のコンポーネントを追加する。そして、それぞれの「Sources」に「世界位置源」を指定してね。

「Position Constraint」の「Weight（全体のウェイト）」を「0.5」に設定する。次に、両方のコンポーネントの「Sources」内にある個別ウェイトを「-1」に設定しよう。

最後に、両方の「Is Active」にチェックを入れて有効化（アクティベート）してね。

#### <2> 「更新位置」のコンストレイント

「更新位置」を選択して、インスペクターで「Parent Constraint」コンポーネントを追加する。「Sources」に「更新位置源」を指定して、最後に有効化（Activate）しよう。

#### <3> 「オブジェクト」のコンストレイント

「オブジェクト」を選択して、インスペクターで「Parent Constraint」コンポーネントを追加する。「Sources」に「更新位置」を指定して、有効化（Activate）してね。

### 3）アニメーションの作成

オフにするアニメーション：「オブジェクト」を非アクティブ（OFF）にして、「更新位置」をアクティブ（ON）にする。

オンにするアニメーション：「オブジェクト」をアクティブ（ON）にして、「更新位置」を非アクティブ（OFF）にする。

### 4）アニメーター（Animator）の設定

Animator に新しくレイヤー（Layer）を作成して、ウェイト（Weight）を「1」にする。パラメーター（Parameter）を新規作成して、タイプは「Bool」にしてね。

そのレイヤー内に、まず「オフ」のアニメーションをドラッグ＆ドロップしてデフォルト状態（Default State）にする。その後、「オン」のアニメーションをドラッグ＆ドロップして、ステート間の遷移（Transition）を作成する。遷移条件（Conditions）にさっき作った Bool パラメーターを設定しよう。
