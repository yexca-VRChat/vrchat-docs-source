> Baseレイヤーを変更する例として、オブジェクトの表示・非表示を切り替えるスイッチを作ってみよう。これが一番自由度が高くて便利なカスタマイズ方法と言えるよ。

## 1. Unityに「Animation」ウィンドウを追加する

画面左上の「Window -> Animation -> Animation」からアニメーションウィンドウを開くことができるよ。好きな場所にドラッグして配置してね（例えば、画面下の「Console（コンソール）」ウィンドウの右側とかがおすすめ）。

## 2. アバターにオブジェクト（小物）を追加する

外部からインポートしたアイテムなら、アバターと同じように「Assets」内にあるprefabファイルやfbxファイルを直接ドラッグ＆ドロップすればOK。

Unity標準のオブジェクト（例えばライトとか）を使う場合は、左側の「Hierarchy（ヒエラルキー）」を右クリックして選ぼう（例えば「Light -> Point Light」とか）。
あるいは、アバター名の上で右クリックして作成すれば、あとは位置を調整するだけで済むよ。

シーンに配置したら、まずはちょうどいい位置に調整しよう。

その後、左側の「Hierarchy」にあるオブジェクトをアバターの「子」要素としてドラッグ＆ドロップする（アバター名の上に重ねてドロップすれば、自動的にアバターの配下に入るよ）。

これで、アバターを動かしたときにオブジェクトも一緒に動くようになるよ。

## 3. オブジェクトを表示（ON）にするアニメーションを録画する

整理しやすくするために、あらかじめアバターのフォルダ内に「Animations」という名前のフォルダを作っておくと便利だよ。「Assets」内でそのフォルダに入っておけば、次にアニメーションを作成するときに自動的にその場所が保存先になるからね。

左側の「Hierarchy」でアバターを選択して、「Animation」ウィンドウにある「Create」ボタンをクリック。保存先を選んで、アニメーションに名前をつけて「保存」をクリックしよう。

録画を始める前に、表示させたいオブジェクトを一旦非表示（オフ）にしておく（例えば、作成したライトをオフにする）。

「Animation」ウィンドウの左側にある「Preview（プレビュー）」の右隣に、赤い丸（録画ボタン）があるから、それをクリックして録画を開始する。

録画が始まったら、表示させたいオブジェクトを表示（オン）に切り替える（例えばライトをオンにする）。

終わったら、もう一度赤い丸をクリックして録画を停止してね。その後、プレビューをオフにして、画面右側の「Inspector（インスペクター）」にあるアバターの「Animator（コントローラー）」を削除しよう（クリックして「Delete」キーを押す）。「Assets」フォルダに戻って、自動生成されたコントローラーも削除して大丈夫。

## 4. アニメーションをFXレイヤーに追加する

### 1）アバターのFXレイヤーコントローラーを開く

コントローラーがある場所を探してクリックすると、「Animator」ウィンドウに表示されるよ（左側でアバター名を選択した後、右側の「VRC Avatar Descriptor」で「FX」レイヤーの部分をクリックすれば場所がすぐ見つかるよ）。

### 2）パラメーターとレイヤーを作成する

「Animator」ウィンドウの「Parameters」タブをクリックして、右側にある「+」をクリック。「Bool」を選択して、名前を変更しよう（例えば「Light」とか）。

次に「Layers」タブをクリックして、右側の「+」をクリック。名前を同じように変更（例えば「Light」）したら、ギアのアイコン（設定）をクリックして「Weight（ウェイト）」を「1」にしよう。

### 3）レイヤーの設定

新しく作ったレイヤーをクリックして、右側の空いている場所で右クリック。「Create State -> Empty」を選んで、そこにさっき録画したアニメーション（例えば「Light」）をドラッグ＆ドロップする。

次に、作成した空（Empty）のステートを右クリックして「Make Transition（遷移の作成）」を選択し、ドラッグ＆ドロップしたアニメーションのステートに持っていって左クリックして繋ぐ。

繋いだ矢印（線）をクリックして、右側の「Inspector」にある「Has Exit Time」のチェックを外す。そして「Conditions」の右下の「+」をクリックして、さっき作ったパラメーター（例えば「Light」）を選択し、値を「true」に設定しよう。

今度は逆に、アニメーションステートから空のステートへ戻る遷移（Make Transition）を作ろう。同じように「Has Exit Time」のチェックを外し、「Conditions」にパラメーター（例えば「Light」）を追加して、値を「false」にする。

## 5. スイッチをメニューに追加する

### 1）パラメーター（Expression Parameters）の追加

Parametersファイルを探してクリックすると、右側のInspectorに表示されるよ（左側でアバター名を選択して、右側の「VRC Avatar Descriptor」の「Parameters」をクリックして場所を確認してね）。

「Add」をクリックして、新しく追加された行に「パラメーター名」と「型」（例えば、名前は「Light」で、型は「Bool」）を入力する。その右側にあるのがデフォルト値で、一番右にあるチェックボックスは「値を保存するかどうか（Saved）」だよ。これにチェックを入れておくと、ゲーム内でライトをONにしたまま別のアバターに着替えて、またこのアバターに戻ってきたときもライトがONのままになるんだ。チェックを外しておくと、着替えたらデフォルト値に戻るよ。

**注意：ここのパラメーター名は、さっきFXレイヤーコントローラーで作ったパラメーター名と完全に一致させる必要があるよ。型も同じにしてね。**

### 2）メニュー（Expressions Menu）の追加

Menuファイルを探してクリックすると、右側のInspectorに表示されるよ（アバターを選択して、「VRC Avatar Descriptor」の「Menu」をクリックして探そう）。

「Add Control」をクリックして名前を入力し、Type（タイプ）を「Toggle」に設定。Parameterにはさっき作ったパラメーター（例えば「Light」）を選択してね。
