> Action レイヤーを編集する例です。アバターの AFK アニメーションを変更します。

## 1. AFK とは

AFK は、VR で VRChat を遊んでいるときにヘッドセットを外すとアバターが取る動作です。デフォルトでは座禅のようなポーズになります。

PC で遊んでいる場合は、“End” キーを押すと AFK 状態に入れます。

## 2. Action レイヤーのコントローラーを開く

コントローラーの場所を見つけてクリックすると、“Animator” ウィンドウに表示されます。左側でアバター名をクリックし、右側の “VRC Avatar Descriptor” から Action レイヤーを探してクリックしても場所を確認できます。

## 3. アニメーションを置き換える

左下の “Afk Init”, “AFK”, “BlendOut” の “Motion” を、使いたいアニメーションに置き換えます。

### 1. 置き換えたいアニメーションファイルが 1 つだけの場合

そのアニメーションを “Afk Init”, “AFK”, “BlendOut” の “Motion” に入れます。

### 2. 置き換えるファイルが 3 つある場合

どれが入場アニメーションで、どれが退出アニメーションかを確認してください。

入場アニメーションは “Afk Init” の “Motion” に入れます。

退出アニメーションは “BlendOut” の “Motion” に入れます。

3 つ目は “AFK” の “Motion” に入れます。
