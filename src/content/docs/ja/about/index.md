## このサイトについて

このサイトは GitHub にデプロイされています。ファイルサイトを作っていたときに Markdown に触れて便利だと思い、その後 Markdown でそのままサイトを作れると知って試してみました。~~今後の更新はここになるかもしれません。~~ WordPress で記事を書くのはかなりつらかったです。~~Markdown に出会わなければ、WordPress に耐えられたのに。~~  
~~以前は両方のサイトを一緒に更新していましたが、こちらのほうが少し早かったかも？~~  
ブログサイト側を変更したことで、形式というか機能がこのサイトと重なったため、サイドバーのスタイルを調整し、各モジュールを前に置きました。

## 記録

ここには、なぜこれらを作ったのかを記録しています。

- きっかけ  
  友人に冗談半分で「チュートリアルを作る」と話したら、とても応援してくれました。  
  正直そのころは知っていることも少なかったので、適当に書いてみようと思っていました。どうせ誰も読まないだろう、と。

- 記事  
  それからブログサイトを作りました。最初の名前は `VRChat教程` でした。制作中に資料を調べているうちに、自分の知識が本当に限られていることに気づき、~~徹夜で~~サイト名を変更しました。

- 動画  
  最初は記事だけを書くつもりでしたが、文章だけでは伝えにくいものもあるので、動画も作ってみました。  
  まさか見てくれる人がいるとは思っていませんでした。それ以降、記事も少しだけ丁寧に書くようになりました。  
  コメントなどを見ると ~~自分が遭遇した問題は少なかったんだな~~ と思いつつ、うれしかったです。  
  その後、さらに深い内容の動画も少しずつ見るようになり、また自分の弱さを実感しました。

- 現在 - 2024.07.31  
  何年も経って、ゲームが更新されたのか、今でも通用するのかは分かりません。ただ当時の私も、当時から見て数年前の記事を読んでいました。詳しい方なら、私の浅い記事から必要な部分を応用してくれるはずです。
