## 1. 準備するもの

オーディオのオンオフを切り替えるアニメーションファイルはあらかじめ録画しておこう。この記事が参考になるよ：[自分でMMDダンスを踊る](/script/self_mmd/#%E4%BA%8C%E5%88%B6%E4%BD%9C%E9%9F%B3%E4%B9%90%E5%BC%80%E5%85%B3)

## 2. パラメーター

| パラメーター | アニメーター(FXレイヤー) | 数値 | 説明 |
| ---------------------------- | ------------- | --------------------------------------------------- | ---------------- |
| スイッチ(Bool) 例：「Music_Start」 | 追加 | 追加 | ランダム再生のオンオフを制御 |
| 曲の選択(Int) 例：「Music」 | 追加 | 任意（追加するとゲーム内で現在の再生曲の表示や次の曲への切り替えが可能） | 選択された曲を制御 |

## 3. レイヤー

## 4. メニュー

VRChatのExpression Parametersにパラメーターを追加したら、メニューを作成するよ。

### 1）スイッチだけ作りたい場合

「Toggle」を一つ追加して、パラメーターに「Music_Start」を設定するだけでOK。

### 2）ゲーム内で再生中の曲を表示したい場合

サブメニューを作るのがおすすめ。

まず「Toggle」を追加して、パラメーターを「Music_Start」に設定する。

次に、曲の数だけ「Toggle」を追加して、パラメーターを「Music」に、値をレイヤー内の対応する条件に設定しよう。

### 3）「次の曲へ」を追加する

実際、ゲーム内で再生中の曲を表示できるようにしておけば、好きな曲を直接クリックして切り替えられるんだ。

だから、単純に「Toggle」を追加して、パラメーターを「Music」、値を「0」か何かに設定しておけば十分だよ。

## 5. その他

このスクリプトを利用すれば、順次再生や逆順再生なども実現できるよ。
