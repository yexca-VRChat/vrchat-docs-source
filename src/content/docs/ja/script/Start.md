> 登場するときは、ちょっと周りと差をつけたいよね

## エフェクト

既成のエフェクトを探して使ってみるのもいいかも。

### エフェクトの処理

注意点として、エフェクトにスクリプト（script）コンポーネントが含まれてちゃダメなんだ。もしあったら削除してね。

スクリプトを消してエフェクトが見栄え悪くなっちゃったら、残念だけど別のを探すしかないかな。

VRChatはスクリプトファイルのアップロードを許可してないからね。

### 実装

エフェクトのループ（Looping）をオフにして、モデルに乗せてアップロードすればOK。

## アイテム

くるくる回るアイテムとかを登場させてみるのはどう？もちろん、エフェクトでもこの方法は使えるよ。

### アニメーション

まずオブジェクトを非表示にしてから、オブジェクトが表示されるアニメーションを録画しよう。

### FX レイヤー

新しいレイヤーを作って、Weight（ウェイト）を1に設定してね。

アニメーションをドラッグ＆ドロップして、デフォルトのアニメーションにする。

次に空の状態（Empty State）を作って、トランジションを作成。Exit Time（終了時間）を設定して、長さは自由に決めてね。（単位は前のステートのアニメーションの長さ。オブジェクトの表示アニメーションや空のアニメーションなら秒単位と考えて大丈夫）

## アニメーションの時間

アニメーションのフレーム数を見れば時間がわかるよ。Unityのデフォルトは1秒60フレーム。

例えば、あるアニメーションが全部で240フレームなら、そのアニメーションは4秒ってことだね。

## モーション

モデルを切り替えた時にトラッカーのリセットが走るから、理論上は「登場モーション」をそのままアニメーションにするのは難しいかも。でも、別のやり方で実現できるよ。 _注：個人的な理解だよ_

### アニメーション

まずはモーションのアニメーションを用意して、次にモデルを非表示にするアニメーション（つまり、モデルのメッシュを全部オフにする）を録画しよう。

### Action レイヤー

#### パラメーター

Bool型のパラメーターを一つ作ろう。例えば `Start` とか。

#### トランジション

`WaitForActionOrAFK` から `Prepare Standing` へ。条件は `Start` が `true`。 `Has Exit Time`（終了時間あり）のチェックは外してね。

#### モーションアニメーション

モーションのアニメーションを放り込んで、 `Prepare Standing` -> `モーションアニメーション` -> `BlendOut Stand` の順でトランジションを繋ぐ。

#### トランジションの条件

`Prepare Standing` -> `モーションアニメーション`： `Start` が `true`、Exit Time なし。

`モーションアニメーション` -> `BlendOut Stand`：Exit Time あり（1に設定）、条件なし。

#### スクリプト

`BlendOut Stand` ステートに `VRC Avatar Parameter Driver` を追加して、パラメーター `Start` を `false` に設定する。

### FX レイヤー

#### パラメーター

Bool型のパラメーターを作る。名前はさっき Action レイヤーで作ったものと同じにするよ（例：`Start`）。

#### レイヤー

新しくレイヤーを作って、Weightを1にする。

以下の `Default` と `Hide` ステートには、録画した「モデル非表示」のアニメーションを割り当てる。 `Start` は空のステートだよ。

レイヤー構造： `Entry` -> `Default` -> `Hide` -> `Start`

#### トランジション

`Default` -> `Hide`：条件なし、Exit Time あり（0.1）。

`Hide` -> `Start`：条件なし、Exit Time あり（0.2）。

#### スクリプト

`Hide` ステートに `VRC Avatar Parameter Driver` を追加して、パラメーター `Start` を `true` に設定。

#### 解説

`Default` -> `Hide` の待ち時間は、モデルのトラッカーリセットが終わるのを待つためのもの。

`Hide` -> `Start` の時間は、モデルが立ちポーズからモーションに移行するまでの予備時間。このあたりは自分で調整してみてね。
