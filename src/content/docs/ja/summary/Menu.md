> VRChatのメニューにはどんな種類があるのかな？

| 種類                       | 紹介                                                |
| :------------------------- | :-------------------------------------------------- |
| Button (ボタン)            | 値が1回だけ変わるよ。だいたい1秒間くらい。           |
| Toggle (トグル)            | クリックすると設定した値になって、オフにするまで保持されるよ。 |
| Sub Menu (サブメニュー)    | 次の階層のメニュー。パラメータの設定は「トグル」と似てるかな。 |
| Two Axis Puppet (2軸パペット) | 2つのパラメータ(float)で垂直と水平を操作するよ（-1.0〜1.0）。 |
| Four Axis Puppet (4軸パペット) | 4つのパラメータ(float)で上下左右を操作するよ（0〜1.0）。しっぽを振る動きとかに使えるね。 |
| Radial Puppet (ラジアルパペット) | 1つのパラメータで操作するよ。だいたい色を変えたり服を着替えたりするのに使うよ。 |

以上がボクの理解。以下は公式の原文だよ。

- **Button** – クリックした時にパラメータをセットして、同期やリセットが送信された後に元に戻るよ。だいたい1秒後くらい。押しっぱなしにはできないんだ。

- **Toggle** – トグルがオンの時にパラメータをセットして、オフにするとリセットされるよ。

- Sub-Menu

  – 別のExpression Menuを開くよ。あと、メニューに入った時にパラメータをセットすることもできて、その場合はメニューを閉じるとパラメータが0に戻るんだ。

  - **大事なポイント:** サブメニューの中にさらにサブメニューを入れることもできるよ！

- **Two Axis Puppet** – ジョイスティックの位置に応じて2つのパラメータを操作するメニューを開くよ。パラメータは垂直と水平に割り当てられてて、float値の範囲は-1.0から1.0までだよ。

- **Four Axis Puppet** – ジョイスティックの位置に応じて4つのパラメータを操作するメニューを開くよ。パラメータは上、右、下、左の順に割り当てられてて、float値は0.0から1.0までだよ。

- **Radial Puppet** – 1つのパラメータを操作するラジアルパペットメニューを開くよ。進捗バーを満たしていくような感じだね！
