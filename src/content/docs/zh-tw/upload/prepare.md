> 從 VRChat 入門到 Unity 精通的道路開啟啦！XD

## 一、前提條件

### 1）擁有 VRChat 帳號（官網註冊）

Steam 登入的不行，如果名字後面有其他字元說明是 steam 登入，請官網註冊帳號並綁定

### 2）帳號等級達到藍色（New User）

線上時間，經常去人多的地方，好友數量，和好友在一起等都會加速升級

## 二、Unity 的下載與安裝

### 1）打開 Unity 官網下載介面

官網下載介面連結 (CN)：[Unity 所有版本下載 - Unity 中國官網](https://unity.cn/releases)

> 條件允許建議下載國際版，但中國版也不是不能用

### 2）下載 Unity Hub 並安裝

下載 Unity 需要先註冊登入才行

### 3）下載 Unity2019.4.31（從 Unity Hub 下載可方便漢化）

如果從 Unity Hub 下載，可以在可選元件裡選擇簡體中文，忘記選擇也沒關係，後面文章會有方法

當前 VRChat 的 SDK 支援的 Unity 版本是 2019.4.31f1，可在此查看：[Currently Supported Unity Version (vrchat.com)](https://docs.vrchat.com/docs/current-unity-version)

## 三、VRChat SDK 的下載

### 1）打開 VRChat SDK 下載介面（需登入）

VRChat 下載介面連結：[Download - VRChat](https://vrchat.com/home/download)

### 2）下載 SDK3.0

點擊“Download SDK3 - Avatars”即可開始下載
注：部分老模型為 SDK2 的，可能需要下載 SDK2(在“Other”裡)，但建議升級 SDK3 上傳（後續文章會介紹）

## 四、必要外掛的下載

### 一、動骨外掛

點擊下載：[Dynamic Bone 1.2.0](https://raw.githubusercontent.com/yexca-VRChat/vrchat-docs-source/main/file/Dynamic%20Bone%201.2.0.zip)

### 二、著色器外掛

點擊下載：[UTS2](https://raw.githubusercontent.com/yexca-VRChat/vrchat-docs-source/main/file/UTS2.zip)

### 三、其他

其他外掛請自行探索，如：[VRC 工具箱](https://www.bilibili.com/video/BV13q4y1f7ZJ)等
