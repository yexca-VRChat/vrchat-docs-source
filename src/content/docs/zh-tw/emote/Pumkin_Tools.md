## Blender 匯出

材質不能修改，與原模型一樣（可能骨骼也不能修改）

## Pumkin's Avatar Tools

GitHub：<https://github.com/rurre/PumkinsAvatarTools>

檔案直連：GitHub V1.1.1：[PumkinsAvatarTools_v1.1.1.unitypackage](https://github.com/rurre/PumkinsAvatarTools/releases/download/1.1.1/PumkinsAvatarTools_v1.1.1.unitypackage)

## 模型處理

將工具與模型匯入 Unity

然後將模型骨骼改為 `Humanoid`，材質為`使用外部材質`，然後可能需要去材質球處選擇貼圖，或使用原模型的材質球

將模型拖入場景，開啟 `Pumkin-Tools-Avatar Tools`

其中 Avatar 選擇模型，在 `Copy Components-Copy From` 選擇預製物

然後點擊 `Copy Selected` 即可
