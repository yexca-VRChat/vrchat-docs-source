> VRChat 的選單有哪些內容呢？

| 類型                       | 介紹                                                |
| :------------------------- | :-------------------------------------------------- |
| Button (按鈕)               | 數值僅變更一次，通常持續約一秒                      |
| Toggle (切換)               | 點擊後會變更為設定值，直到關閉為止                  |
| Sub Menu (子選單)           | 下一層選單，設定參數的方式類似「Toggle」            |
| Two Axis Puppet (雙軸控制)  | 兩個參數 (float) 控制垂直與水平 (-1.0~1.0)          |
| Four Axis Puppet (四軸控制) | 四個參數 (float) 控制上下左右 (0~1.0)，可用來製作搖尾巴等效果 |
| Radial Puppet (百分比轉盤)  | 由單一參數控制，通常用來改變顏色或更換衣服          |

以上為個人理解，以下為官方原文

- **Button** – 點擊時設定參數，接著在同步/重置發送後重置——通常在約一秒後。無法長按。

- **Toggle** – 當切換開關開啟時設定參數，當切換開關關閉時重置。

- Sub-Menu

  – 開啟另一個 Expression Menu（表情選單）。此外，進入時也可能設定一個參數，若是如此，當你退出該選單時，該參數將重置為零。

  - **重要提示：** 你可以將子選單放入子選單中！

- **Two Axis Puppet** – 開啟一個根據搖桿位置控制兩個參數的軸向傀儡選單。參數會映射到垂直與水平。浮點值 (float) 範圍從 -1.0 到 1.0。

- **Four Axis Puppet** – 開啟一個根據搖桿位置控制四個參數的軸向傀儡選單。參數依序映射為：上、右、下、左。浮點值 (float) 為 0.0 到 1.0。

- **Radial Puppet** – 開啟一個控制單一參數的徑向傀儡選單，有點像是一個你可以填滿的進度條！
